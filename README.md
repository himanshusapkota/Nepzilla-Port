# 🚀 Himanshu Sapkota Portfolio Website

A modern, responsive, and interactive personal portfolio website built using **HTML, CSS, and JavaScript**. This portfolio showcases my skills, projects, services, experience, and contact information while providing a smooth and engaging user experience.

## 🌟 Features

* Modern and responsive design
* Dark/Light theme toggle
* Animated loading screen
* Custom cursor effects
* Scroll progress indicator
* Mobile-friendly navigation menu
* Typing text animation
* Interactive skills section
* Dynamic projects showcase
* Services section
* Experience timeline
* Contact form with toast notifications
* Social media integration
* Smooth scrolling animations
* Fully responsive across devices

## 🛠️ Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript (Vanilla JS)

### Libraries & Resources

* Font Awesome Icons
* Google Fonts

  * Syne
  * DM Sans
